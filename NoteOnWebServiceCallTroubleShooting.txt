To isolate a SOAP service request:
Look for the service end point in tsh common.properties:
	iamap.services.geocode.endpoint=http://dev.virtualearth.net/webservices/v1/geocodeservice/geocodeservice.svc
	iamap.services.geocode.type=net.virtualearth.dev.webservices.v1.geocode.BasicHttpBinding_IGeocodeServiceStub
	
Enable log4j properties to debug on all apache packages.

Run tests.

From log pluck the request corresponding to the service defined in property file which is webservices/v1/geocodeservice/geocodeservice.svc
